Core modules

In the default Akkamo distribution, several modules are available, both as a technology demo and a fully working modules. Here is the list of available modules:

  • Akka module - provides support for the Akka actor framework
  • Akka HTTP module - provides support for the Akka HTTP, a full server-and client-side HTTP stack on top of akka-actor and akka-stream libraries
  • Config module - provides application-wide configuration, based on the Lightbend Config library
  • Kafka module - provides support for the Apache Kafka, publish-subscribe messaging rethought as a distributed commit log
  • Log module - providing logging support
  • Persistent config module - providing persistable configuration, based on the Lightbend Config and by default backed by the MongoDB database
  • ReactiveMongo module - provides support for the MongoDB database, using the ReactiveMongo Scala driver
